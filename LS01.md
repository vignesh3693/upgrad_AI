## 1Ô∏è‚É£1Ô∏è‚É£ Object-Oriented Programming (OOP)

### Class Example

```python
class BankAccount:
    def __init__(self, balance):
        self.balance = balance

    def deposit(self, amount):
        self.balance += amount
```

### Object Creation

```python
acc = BankAccount(1000)
acc.deposit(500)
```

### Key Concepts

* Class = blueprint
* Object = instance
* `self` = current object
